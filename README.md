# iDiscuss - Coding Forums

iDiscuss is a vibrant online community dedicated to discussing various topics related to coding and programming. Whether you're a seasoned developer or just starting your coding journey, iDiscuss provides a platform for sharing knowledge, seeking help, and engaging in discussions with like-minded individuals.

## Key Features

- **Thread Creation:** Users can create new discussion threads on topics of their interest. Each thread consists of a title and a description, providing context for the discussion.

- **Search Functionality:** The forum includes a powerful search feature that allows users to search for specific threads based on keywords. The search results display relevant threads matching the user's query.

- **Responsive Design:** iDiscuss is built using Bootstrap, ensuring a responsive and mobile-friendly user interface. Users can access the forum seamlessly across various devices, including desktops, tablets, and smartphones.

- **User Authentication:** To participate in discussions and create new threads, users need to log in to their accounts. User authentication helps maintain the integrity of the forum and enables personalized interactions.

- **Commenting System:** Users can engage in discussions by posting comments on existing threads. The commenting system allows for threaded discussions, facilitating clear and organized conversations.

- **Community Guidelines:** iDiscuss promotes a positive and respectful community environment. Users are encouraged to adhere to community guidelines, which prohibit spam, offensive content, and copyright infringement.

## Usage

1. Clone the repository to your local machine.
   ```bash
   git clone https://github.com/your-username/iDiscuss.git


2. **Import Database Schema:** Import the database schema located in the `database.sql` file to set up the required database tables.

3. **Update Database Connection:** Update the database connection settings in the `_dbconnect.php` file with your MySQL database credentials.

4. **Start Local Server:** Start a local server (e.g., using XAMPP, WAMP, or MAMP) and navigate to the project directory in your web browser.

5. **Ready to Use:** You're ready to use iDiscuss! Create an account, start new threads, join discussions, and explore the coding community.
